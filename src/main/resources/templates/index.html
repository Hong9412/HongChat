프로젝트 설명


소켓통신(STOMP)을 이용한 실시간 채팅 프로그램입니다.

주요 기능

- 채팅방 생성 -> 채팅방의 이름, 인원수, 비밀번호를 설정할 수 있습니다.
- 나의 채팅방 -> 내가 속한 채팅방을 조회할 수 있습니다.
- 전체 채팅방 -> 등록된 채팅방을 모두 조회할 수 있습니다.


채팅방

- 방에 속한 유저리스트 정보와, 실시간 채팅을 이용할 수 있습니다
- 채팅 내용은 DB에 저장되고 각 유저마다 입장한 시점부터 저장된 메세지를 확인할 수 있습니다.
- 방 생성자라면 해당 방에 속한 유저를 강제 퇴장 시킬 수 있습니다.
